# 步骤一：中外跨境电商供应链绩效对比

# 教学内容：

首先，让学生了解供应链管理进程模型，学习最基础、应用最广泛的SCOR模型，通过对计划、采购、生产、配送和退货五个流程的讲解，以及四层结构的分析，了解最基本的供应链模型。其次，让学生熟悉跨境电商供应链绩效评价模型，掌握供应链绩效的五个基本驱动因素和三个辅助驱动因素。最后，了解跨境电商供应链绩效的财务指标，以亚马逊和京东的财务绩效作为比较案例进行讨论分析，了解两大跨境电商龙头公司如何实现绩效的评价。

# 实验操作：

京东、亚马逊2017年和2018年财务数据比较分析。

# 实验效果：

学生通过自主学习，查询京东和亚马逊2015和2016年财务数据，进行讨论分析，促进了对跨境电商供应链绩效的理解，也可以引发学生对供应链绩效的思考，促进了学生继续深入学习跨境电商供应链的兴趣。
