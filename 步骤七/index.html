# 步骤七：利用综合计划模型求解生产投入关键变量

# 教学内容：

讲授跨境电商供应链的综合计划，了解跨境电商供应链综合计划的概念、特征及作用。然后以暖冬公司为例来介绍线性规划的应用，从确定总量单位、需求预测、成本预测、确定约束条件，到确定计划变量、设置约束条件，最后通过确定目标函数，利用综合计划模型求解。

# 实验操作：

以暖冬公司为例，利用综合计划模型进行成本最小化求解。 暖冬公司是一家羽绒服生产商，其工厂位于美国。暖冬公司的产品通过电商以及线下零售的方式在美国及海外进行跨境电子销售。由于进行生产的设备和场地等条件较容易满足，因此，暖冬公司的产能主要由员工数量决定。在本例中，我们假定计划期为6个月，因为在6个月的时间里足以说明综合计划中的很多问题。

顾客对暖冬公司生产的羽绒服需求具有高度的季节性。旺季当然是寒冷的冬季，人们需要优质的羽绒服来保暖。这一季节性需求波动在供应链上从下游的消费者传递到制造商暖冬公司。公司应对需求季节性波动的方法包括在旺季增加产能（劳动力规模或机器规模）、在淡季积累库存、将一定数量的产品转包或者由于供不应求放弃一部分订单。

为了合理的完成综合计划制定，暖冬公司需要首先完成总量单位的确定、需求预测、成本预测和约束条件的确定。该公司在其制造厂生产6个产品系列。每个产品系列的生产时间、调试时间、成本、收入和批量生产记录如下所示。根据正常生产时间和调试的分摊时间，可以得出每单位的净生产时间。因此，系列A的每单位净生产时间为6/80+0.8=0.875小时。
