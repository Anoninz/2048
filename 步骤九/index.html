# 步骤九：周转库存决策

# 教学内容：

讲授跨境电商供应链的库存管理，让学生了解跨境电商供应链库存的概念，熟悉库存的分类，同时掌握跨境电商供应链库存管理的决策方法，包括跨境电商周转库存决策。

# 实验操作：

熟悉跨境电商供应链库存管理的主要内容，学会简单计算周转库存的成本以及无批量折扣性质的周转库存。

假设某销售电脑的电商每个月的需求量为1000台。每次订货的固定订购、运输和收货成本为4000美元。电商每台电脑的进货价格为500美元，库存持有成本为20%。计算电商每次补货时应当订购多少台电脑？周转库存是多少？

如果销售电脑的电商希望将最优订货批量由980台降低到200台。要想使该订货批量成为最优，每批订货成本应当降低到多少？

# 实验效果：

通过对库存管理的介绍，让学生懂得库存对市场发展、企业的正常运作起着非常重要的作用，掌握了周转库存相关的基础知识。
