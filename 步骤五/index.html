# 步骤五：构建网络优化模型并进行求解

# 教学内容：

介绍《跨境电商供应链管理》第五章第三节内容，通过实例让学生熟悉学过的网络优化模型。通过建立一个简化的四级供应链模型，对工厂和仓库进行选址，以实现整条供应链成本最小为目标，以网络优化模型为基础，利用Excel的Solver工具来对模型进行求解。

# 实验效果：

通过建立网络优化模型，分别计算出原材料成本、生产成本、总关税成本以及总运输成本，通过决策变量的变化，以Solver工具求得最优解。将理论转化为实践，真正学会了如何进行网络优化模型的建立。
