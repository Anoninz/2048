# 步骤六：利用重力选址模型为工厂和仓库选址

# 教学内容：

以上一节课利用网络优化模型得出公司设施区域位置的基础上，利用重力选址模型来确定位于东南亚并服务当地市场的工厂和仓库的潜在位置。通过变换相应的假设条件，让学生继续熟悉重力选址模型。

以步骤五实例为基础，并假设在东南亚平均每一万台手机从工厂到仓库的平均运输成本为每公里1.5元，具体如下表所示。构建重力选址模型求解工厂和仓库的坐标。

# 实验效果：

计算出重力选址的决策变量，以其位基础，利用Solver工具求得最优解。通过类似于网络优化模型的操作步骤，让学生熟悉了怎样利用重力选址模型预测最优解，同时可以让他们体会到两种预测模型的不同，有利于在以后的实际操作中合理选择预测模型进行预测。
