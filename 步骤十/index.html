# 步骤十：有批量折扣和考虑商业促销下的周转库存决策

# 教学内容：

在上一堂课掌握了周转库存相关基础知识和简单的成本计算方法的基础上，重点讲授了有批量折扣性质的周转库存和考虑商业促销情形下的周转库存的优化问题。

# 实验操作：

让学生理解影响周转库存的各项因素，学会计算各项条件下的周转库存，包括有平均批量折扣或边际批量折扣性质的周转库存、考虑商业促销情形下的周转库存。

首先，某在线销售耳机的跨境电商，每个月耳机的需求量是10000条，该电商每次向制造商订购耳机时都要承担100美元的固定订货、运输和收货成本，电商的库存持有成本为20%。制造商采用的全部单位数量折扣定价方案为，订货量在0~4999条耳机时，单位定价为3美元；订货量在5000~9999条耳机时，单位定价为2.96美元；订货量在10000或以上时，单位定价为2.92美元。计算最优订购批量是多少？

其次，我们还是以上例为例，与之不同的是，制造商采用边际单位折扣定价方案，订货量在0~4999条耳机时，单位定价3.00美元；订货量在5000~9999时，超过5000的部分按照2.96美元，其余的5000条依然按照3.00美元的单价支付；订货量在10000或以上时，超过10000的部分按照2.92美元支付，其余的分别按照各自的划分单价来支付。这意味着，如果订购8000条耳机时，最初的5000条的单位成本是3.00美元，剩下的3000条的单位成本是2.96美元。这时计算该跨境电商在每个订货批次中应当订购多少条耳机？

再次，假设该耳机的年需求量是120000条，即每月需求为10000条，制造商对每条耳机的售价为3美元，该跨境电商的库存持有成本为20%，制造商不提供任何折扣时的最优订货批量6324条。制造商对接下来的一个月跨境电商所购买的所有耳机提供每条0.15美元的折扣，在促销期间，该跨境电商应当购买多少条耳机？

最后，假设该跨境电商所面临的需求曲线为300000-60000p。制造商向跨境电商收取的正常价格为3美元/条，不考虑与库存相关的所有成本，跨境电商在折扣为每条0.15美元时的最优反应是什么？

# 实验效果：

学生通过对立体的深度剖析，在学会计算方法的基础上真正理解了周转库存对供应链的重要性以及影响周转库存的因素。
