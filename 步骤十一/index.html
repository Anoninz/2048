# 步骤十一：安全库存决策

# 教学内容：

在讲解安全库存的概念、安全库存的作用、安全库存合理水平的影响因素等安全库存相关基础知识的基础上，讲授了基本的安全库存决策流程与计算方法。

# 实验操作：

首先，假设某销售智能手机的跨境电商的周需求服从正态分布，均值为3000，标准差为500.制造商要花2周时间才能满足该跨境电商的订单。目前，当库存里的智能手机数量降到8000部时，采购经理会向制造商订购10000部智能手机。计算该跨境电商的安全库存和平均库存，并计算每部手机在库存的平均存放时间。

其次，销售智能手机的跨境电商面临的周需求服从正态分布，均值为2500，标准差为500，补货提前期为2周。假设每周的需求相互独立，当智能手机的库存降到6000部时，采购经理会向制造商订购10000部智能手机，计算该策略下跨境电商的CSL。

最后，假设其面临的周需求服从正态分布，均值为2500，标准差为500，补货提前期为2周，并且假设每周的需求相互独立。当库存内智能手机的存量降到6000部时，采购经理会向制造商订购10000部智能手机，计算该策略下跨境电商的满足率。

通过上面三个案例分别计算了给定补货策略下周期服务水平和满足率、给定周期服务水平和满足率下的安全库存以及服务水平可变情况下的安全库存。

# 实验效果：

学生通过对安全库存基础知识以及三种条件下不同计算方法的学习和练习，对供应链管理中安全库存方面的问题有了较为深入的理解。
