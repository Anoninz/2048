# 步骤十三：跨境电商供应链管理系统操作

# 教学内容：

让学生通过实际操作来学会在WMS系统中创建公司、维护商家信息和资源分配、入库和销售订单、仓库作业、计费和结算、进行报关操作等。

# 实验操作：

模拟在WMS系统中创建公司。 模拟在WMS系统中维护商家信息和资源分配。 模拟在WMS系统中入库和销售订单。 模拟在WMS系统中进行仓库作业。 模拟在WMS系统中计费和结算。 模拟在WMS系统中进行报关操作等。

# 实验效果：

通过实际操作，学生学会如何在WMS系统中创建公司、维护商家信息和资源分配、入库和销售订单、仓库作业、计费和结算、进行报关操作等。
