# 步骤十二：存储、最优可获得性水平、补货策略安全库存决策

# 教学内容：

在上节课讲授安全库存相关基础知识以及几种条件下安全库存的决策流程及方法的基础上讲解了其余几种条件下安全库存决策的影响因素、分析过程与计算方法。

# 实验操作：

首先，某电商的美国分公司正在考虑两种方案：其一，在美国的东、南、西、北四个地区各设置一个配送中心；其二，仅在纽约设置一个中心，服务于全国。假设其销售的产品的周需求服从正态分布，均值为1000单位，标准差为300单位。虽然每个地区的需求都是独立的，但是供应提前期差不多相同，都是4周。假设其产品的成本是1000美元，库存持有成本为20%。由于承诺次日送达，美国分公司必须为四个地区中心承担全部的内陆运输费用10美元/单位。不过，如果在全美国仅设置一个配送中心，则必须组建一个成本更加高昂的运输车队，次日送达的服务费将是13美元/单位。设置并运营四个区域配送中心每年的费用是15万美元，比仅在纽约设置一个全国配送中心所需的建设和运营费用要高得多。假设该电商期望的CSL是0.95。基于上述的费用进行考虑，美国分公司将会采取什么样的决策？

其次，假设该电商销售的产品之一是冲锋衣。冲锋衣的销售季节是11月至次年2月，该电商的采购经理现行的做法是在销售季节到来之前，从制造商那里购入整个销售季节所需的冲锋衣。根据过去几年的市场需求情况，采购经理估计冲锋衣的需求分布，如下表所示，这不同于采用需求的历史数据的平均值作为共识预测的传统做法。为了便于讨论，我们假设所有的需求量都以百为单位，制造商也要求电商以100的倍数为批量进行采购。根据这些条件计算最优订货量？
